Hreflang

HREFLANG permite indicar qual é o idioma principal do site para onde o link está desviando o fluxo de navegação

< a href="https://www.w3schools.com/html/" hreflang="en">Site da w3schools (em inglês)</a>

Atributo target

Para que a pagina seja aberta podemos utilizar o atributo TARGET como os valores:

  • _blank vai abrir o link em uma nova janela
  • _self vai abrir o link na janela atual(padrão)
  • _top vai se desfazer de todos os frames e abrir o destino no navegador completo
  • _parent similar ao uso do _top em uma referência a janela mãe

Como o uso de frames é uma técnica quase em desususo, vamos nos basear apenas nas duas primeiras opções _self e _blank.
